# UniGo Eltish xizmati qo'shilgan qismlar

## Qo'shilgan logika
- Shahar ichida eltish
- Tumanlar aro eltish
- Viloyatlar aro eltish
- Pickup: standart nuqta yoki aniq xarita
- Dropoff: standart nuqta yoki aniq xarita
- Buyum turi: hujjat, kalit, kichik buyum, buyum
- Buyum uchun kg limiti
- Viloyatlar aro reys bilan match qilish
- Haydovchi panelida status boshqarish

## Tavsiya etilgan DB
Yuqoridagi SQL ni Supabase SQL Editor ga qo'ying.

## Eslatma
Agar `delivery_orders` jadvali hali yaratilmagan bo'lsa, front-end localStorage fallback bilan ishlaydi.
Bu demo/tekshiruv uchun qulay, lekin production uchun SQL ni qo'yish kerak.
