# UniGo Driver App (React Native + Expo)

## Boshlash
```bash
npm install
npm run start      # Expo Go
npm run android    # Android emulator
npm run ios        # iOS simulator
```

## Asosiy funksiyalar
- Haydovchi online/offline toggling
- Real-time GPS tracking (har 3 soniya)
- Yangi buyurtma radar (5km radius)
- Kunlik daromad hisobi
